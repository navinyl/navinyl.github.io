The NVSTYLE Project

I did this for fun, if you want to use this as your main os, its your choice, you will need to do a clean install.

You will need a USB drive of at least 8GB or more, Use Rufus or Ventoy to flash.

Changelog

- NVSTYLE is a Windows 11 mod that deshitifies Windows 11 and makes it better.

- Based off Windows 11 IoT LTSC with June updates

- Uses Windows 10 IoT LTSC setup as installation

- Debloated

- Uses ExplorerPatcher as the shell cuz windows 11's shell sucks (you can uninstall it if you dont like the shell)

- Pin everywhere!

- Heavily tweaked, uses few processes and only 1.9GB of ram

- Size: 5.88GB
